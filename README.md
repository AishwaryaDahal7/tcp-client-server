# TCP & UDP Client-Server Applications

Basic Unix client-server implementations in C for TCP and UDP protocols.

## Branches

- `tcp-implementation` — TCP client-server application
- `udp-implementation` — UDP client-server application

## Build

```sh
make
```

## Usage

**TCP:** `./tcp_server` & `./tcp_client [message]`

**UDP:** `./udp_server` & `./udp_client [message]`
